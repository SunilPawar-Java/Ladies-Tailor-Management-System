**#Tailor Management System**

A full-stack web application for managing customer measurements, orders, and custom clothing stitching.

**Overview**
The Tailor Management System is a web application designed to digitize tailoring operations. 
It allows customers to place custom clothing orders, manage measurements, track order status, and securely make payments.
The application provides a RESTful backend using Spring Boot and a responsive frontend using React.

**Features**
- User Registration & Login
- JWT Authentication
- Customer Measurement Management
- Product Management
- Cart Functionality
- Order Placement
- Payment Integration
- Order Tracking
- Customer And Admin Dashboards
- Role-Based Authorization
